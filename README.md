# 🎯 Hangman Game in Python

A clean and interactive terminal-based Hangman Game built with Python.

This project demonstrates core Python programming concepts through a classic word-guessing game where the player must discover the hidden word before the hangman is fully drawn.

It is designed to showcase problem-solving skills, game logic implementation, clean code structure, and beginner-to-intermediate Python development practices.

---

# 📌 Project Overview

The Hangman Game is a word guessing game where:

* A random word is selected from a large predefined word bank
* The player guesses one letter at a time
* Correct guesses reveal the hidden letters
* Wrong guesses add parts to the hangman drawing
* The game ends when the player either guesses the full word or reaches the maximum number of wrong attempts

This project focuses on writing clean, readable, and structured Python code while implementing a complete playable game.
